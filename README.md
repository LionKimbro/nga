# Nga: a Stack Oriented Virtual Machine

## Overview

Nga is a lightweight, stack based virtual machine.

It features:

* Small core (under 300 lines of C code)
* Dual stack, MISC instruction set
* 27 instructions
* No defined I/O model
* Extensible design
